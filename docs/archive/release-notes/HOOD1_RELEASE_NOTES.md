# HOOD1 — бытовые кухонные вытяжки

Релиз добавляет отдельный B2C-кластер бытовых вытяжек и не изменяет семантику ресторанной вентиляции.

## Состав

- 15 страниц: хаб, диагностика, 8 симптомных сценариев и 5 сервисных работ.
- 60 контекстных ссылок внутри кластера.
- Ссылки из `bytovaya-index.html`, `bytovaya-uslugi.html` и `plity.html`.
- Разделение режима отвода в канал и рециркуляции через угольный фильтр.
- Evidence-реестр, таксономия, intent boundaries, screenshot contract и guard-команда.

## Основные запреты

- Слабая тяга не означает автоматически неисправный двигатель.
- Шум не означает автоматически повреждённые подшипники.
- Запрещены советы по самостоятельному ремонту двигателя, платы и сетевой проводки.
- Бытовые страницы не подменяют ресторанные зонты и промышленную вентиляцию.
- Эксплуатация без предусмотренного жирового фильтра не рекомендуется.
