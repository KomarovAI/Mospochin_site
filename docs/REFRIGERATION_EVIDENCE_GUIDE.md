# Refrigeration evidence guide

Checked: 2026-07-15

Allowed sources: official manufacturer pages, official catalogues, official user/service manuals, official controller manuals and official technical guides.

Every record must define equipment/model scope, supported claims and prohibited generalisations. Controller codes are controller-scoped; they are not brand-wide appliance codes. Refrigerant, pressure, electrical and cycle values require a specific model document.

Current foundation uses official Polair, Abat, HICOLD, Liebherr, IRINOX, Coldline, Hoshizaki, Scotsman, Danfoss and Copeland sources.
