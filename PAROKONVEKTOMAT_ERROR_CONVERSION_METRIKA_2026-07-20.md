# Цели Метрики для конверсионного слоя ошибок пароконвектоматов

## Цели, которые должен принимать счётчик
1. `error_call_click` — нажатие звонкового CTA на выбранных инженерных страницах.
2. `error_photo_whatsapp_click` — переход в WhatsApp с контекстом кода.
3. `error_callback_submit` — отправка формы обратного звонка.
4. `error_photo_submit` — отправка формы предварительного разбора.
5. Существующие `phone_click`, `whatsapp_click`, `form_submit_success` — контрольные общесайтовые цели.

## Рекомендуемые приоритеты для Директа
- подтверждённый уникальный целевой звонок — максимальная ценность;
- успешная форма обратного звонка — высокая;
- WhatsApp с фото и моделью — высокая/средняя после проверки качества;
- клик по номеру — вспомогательная микроконверсия, не основная цель оптимизации.

## Важно
Клик по `tel:` не подтверждает состоявшийся разговор. Для целей «звонок» и «целевой звонок» данные должен передавать подключённый коллтрекинг. До подключения коллтрекинга кампанию нельзя оптимизировать только по `phone_click`.

## Официальные инструкции
- Клики по телефонам: https://yandex.ru/support/metrica/en/simple-goal/tel
- Данные звонков: https://yandex.ru/support/metrica/en/data/calls
- Цели звонков: https://yandex.ru/support/metrica/en/general/call-goal
- Релевантность объявления и посадочной: https://yandex.ru/support/direct/en/general/positions
