# PAROKONVEKTOMAT ERROR NAVIGATION — CHANGELOG

Дата: 2026-07-20

## Реализовано

- Единая навигационная модель для 59 инженерных страниц.
- Унифицированные HTML breadcrumbs и BreadcrumbList JSON-LD.
- Постоянный навигатор по производителям, функциональным контурам и последствиям.
- Для 41 карточки кодов: производитель, серия, контур, последствие, предыдущая/следующая карточка и оглавление.
- Блок «Куда перейти дальше» с кодами той же системы, симптомами и сервисными маршрутами.
- Двусторонние ссылки из симптомных и сервисных страниц в инженерный справочник.
- Hash-фильтры центрального поиска без создания индексируемых комбинаций URL.
- События навигации для аналитики через data-site-event и error-navigation.js.

## Данные

- `data/parokonvektomat-error-navigation.json`
- 59 страниц кластера
- 41 карточка кодов/групп
- 20 связанных симптомных и сервисных страниц
