СПЕЦИФИКА ПО БРЕНДАМ

Особенности ТО для каждого бренда

У каждого производителя свои регламенты и типичные болячки. Мы знаем их все.

Rational SCC / iCombi

  • Декальцинация каждые 3 месяца
  • Проверка iLevelControl
  • Калибровка датчика Core Temp
  • Чистка парогенератора

Типичные ошибки: E05, E09, E15

Unox BakerTop / ChefTop

  • Декальцинация при ошибке CLEAN
  • Чистка форсунки впрыска воды
  • Проверка панели TOUCH/LED
  • Чистка дренажной системы

Типичные ошибки: AF02, AF08, CLEAN

Abat / Абат ПКА

  • Чистка бойлера парогенератора
  • Проверка насоса подачи воды
  • Чистка дренажа
  • Проверка ТЭНов

Типичные ошибки: E02, E07, E10

Convotherm C4 / C6

  • Декальцинация нагревательных элементов
  • Проверка панели easyTouch/easyDial
  • Чистка инжектора пара
  • Проверка датчика влажности

Типичные ошибки: Err 3, Err 7

Lainox Naboo / Krea

  • Чистка фильтра водоподготовки
  • Проверка соленоидного клапана
  • Проверка датчика уровня воды
  • Чистка парогенератора

Типичные ошибки: E04, E08

Electrolux Air-O-Steam

  • Чистка бойлера от накипи
  • Проверка датчика давления пара
  • Проверка системы оптимизации воды
  • Чистка Unity-панели

Типичные ошибки: ErW, ErP
