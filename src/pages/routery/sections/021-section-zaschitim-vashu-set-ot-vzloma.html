Смена пароля админки

admin/admin — открывает доступ к настройкам. Ставим сложный пароль, включаем 2FA если есть.

WPA3 шифрование

WPA2 — уже уязвим. WPA3 — новый стандарт. Настраиваем максимальное шифрование.

Скрытие SSID

Скрываем имя сети от соседей. Подключаем устройства вручную — никто не увидит вашу сеть.

Гостевая сеть

Отдельная сеть для гостей — без доступа к вашим устройствам, принтерам, камерам.

Родительский контроль

Блокировка сайтов, ограничение по времени, фильтрация контента для детей.

Брандмауэр (Firewall)
