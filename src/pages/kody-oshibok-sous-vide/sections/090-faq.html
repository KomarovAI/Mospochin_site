Частые вопросы по этому симптому

Почему нельзя использовать универсальную таблицу кодов?

Одинаковая комбинация символов может иметь разный смысл у разных брендов и даже серий. Значение публикуется только вместе с модельной областью.

Какие коды Anova уже подтверждены?

Для Precision Cooker Nano подтверждены LO, STOP и ----; для Precision Cooker Pro — LO и ----. Значения и действия указаны на странице по модельным строкам.

Нужно ли перезапускать аппарат при любом коде?

Нет. Если производитель требует прекратить использование, перезапуск недопустим. Сначала определяют модель и официальное значение сообщения.
