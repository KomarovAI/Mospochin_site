Частые вопросы

Можно ли оценивать цикл только по температуре воздуха?

Нет. Для программы по щупу ключевой параметр — температура в центре продукта; воздух камеры может быть значительно холоднее.

Медленное охлаждение всегда означает нехватку хладагента?

Нет. Сначала проверяют загрузку, термощуп, вентиляторы, испаритель, конденсатор и настройки программы.

Есть ли единое нормативное время для любого шокера?

Нет. Время зависит от модели, массы, геометрии и начальной температуры продукта; паспортные значения применимы только в заданных условиях.
