# Визуалы сервисного архива

Файлы в этой папке подготовлены из `combi_oven_visual_archive`: hero-сцены и улучшенные сервисные фотографии для ресторанного оборудования.

## Правила публикации

- Hero-сцены использовать как иллюстрации услуги, не как фото конкретного выезда или отзыва клиента.
- Кадры с платами, проводкой, снятыми панелями, загрязнением и коррозией подписывать как диагностику, «до ремонта» или «в процессе».
- Не выдавать изображения за точную фиксацию состояния конкретной модели и не использовать их как электрическую схему.
- Для брендов ABAT, UNOX и EKA сохранять контекст конкретной страницы бренда; не заявлять официальное представительство.
- На публичных кейсах отдельно проверять маркировки, серийные номера и шильдики.

Responsive-варианты и WebP создаются штатными image-скриптами проекта.
