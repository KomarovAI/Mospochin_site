## 🎯 Рекомендации для будущих hero-картинок

**Что ищем:**
- Техника с видимыми признаками поломки (таблички, открытые панели, инструменты)
- Мастер за работой (диагностика, ремонт)
- Эмоция проблемы ("не работает", "сломалось")
- Бытовая техника: стиралки, холодильники, посудомойки, плиты

**Что НЕ ищем:**
- Автосервис, автомобили
- Прачечные, стирка
- Медицинское оборудование
- Новая техника, интерьеры
